# Changelog

Все изменения в проекте "Админ-Ко" будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

## [Unreleased]

### Fixed
- **Test Logic**: Исправлена ошибка, при которой ответы на вопросы теста не вызывали реакции бота. Добавлена обработка "зависших" состояний (автоматический переход к актуальному вопросу).
- **Formatting**: Все тексты в `texts.py` переведены с Markdown на HTML разметку (<b> вместо **, <code> вместо `) для корректной работы с `parse_mode='HTML'`.
- **Stability**: Добавлена обработка исключений (try-except) и логирование в основные обработчики теста для предотвращения "тихих" падений.
- **Bot Conflict**: Устранена проблема запуска нескольких инстансов бота, приводившая к конфликту `getUpdates`.

### Added
- **Miro Integration**: Полная двусторонняя интеграция с Miro.
    - Скрипт `miro/export.py` для создания полной визуальной карты бота (сообщения, кнопки, связи).
    - Скрипт `miro/manager.py` для чтения состояния доски и анализа изменений.
    - Документация в `miro/README.md`.
- **Visualization**: Добавлена поддержка полных текстов сообщений и всплывающих окон (Alerts) на диаграмме.
- **Tools**: Скрипт `tools/docs.py` для автоматического ведения чейнджлога.

### Changed
- **Documentation**: Полностью переписан основной `README.md`. Акцент смещен на визуальное управление через Miro.
- **Cleanup**: Удалена устаревшая папка `mermaid/`, так как проект перешел на Miro API v2.
- **Architecture**: Структура проекта обновлена для поддержки модульности (отдельные папки для инструментов).

### Fixed
- Исправлена ошибка `405 Method Not Supported` при работе с Miro API v2 (замена `/items` на `/shapes`).
- Исправлен параметр цвета (`backgroundColor` -> `fillColor`) для совместимости с новым API Miro.
